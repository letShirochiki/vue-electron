# vue3 + typescript + element + sass + electron

## 安装依赖包
```
npm install
```

### web端启动
```
npm run serve
```

### web端打包
```
npm run build
```

### 桌面端启动
```
npm run electron:serve
```

### 桌面端打包
```
npm run electron:electron
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

### Customize configuration
See [vue-cli](https://cli.vuejs.org/config/)<br>
